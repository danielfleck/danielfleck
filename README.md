## 🔐 Dados, segurança e boas práticas

Tenho preocupação explícita com segurança, privacidade e uso responsável de dados.

Nos meus repositórios públicos, adoto como regra:

- utilizar dados fictícios, públicos ou anonimizados;
- não publicar credenciais, arquivos `.env`, tokens ou informações restritas;
- não versionar dados sensíveis, dependências geradas ou artefatos locais;
- usar `.gitignore` de forma preventiva;
- documentar quando os dados forem sintéticos;
- separar exemplos didáticos de qualquer dado real.

Também tenho interesse em práticas de anonimização, mascaramento e proteção de dados, especialmente em projetos envolvendo relatórios, indicadores, bases operacionais e automação.
